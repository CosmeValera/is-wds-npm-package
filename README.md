# is-wds-csm

First attempt to publish a npm package.

### Test npm package locally:
```bash
npm link
```
## Publishing it to the web

### Log in your npm account
```bash
npm login
```

### Check you are logged in:
```bash
npm whoami
```

### Publish package:
```bash
npm publish
```
